# Jargon Rewrite Lexicon

Use this file when the input includes business buzzwords, abstract stacking, or filler-heavy Chinese.

## Common Replacements

Replace with context-appropriate plain Chinese.

- 落 -> 帮助 / 支持
- 落盘 -> 保存
- 抓手 -> 办法 / 关键措施
- 闭环 -> 完整流程 / 完成并反馈
- 对齐 -> 达成一致 / 统一理解
- 拉通 -> 协调打通 / 统一推进
- 提效 -> 提高效率
- 降本增效 -> 降低成本并提高效率
- 沉淀 -> 积累 / 形成
- 打磨 -> 改进 / 优化
- 链路 -> 流程 / 路径
- 颗粒度 -> 细化程度
- 拆解 -> 分解
- 跑通 -> 验证可行 / 实现
- 覆盖 -> 包含 / 涉及
- 触达 -> 联系到 / 影响到
- 复盘 -> 回顾总结
- 矩阵 -> 一组渠道 / 一组账号
- 场景化 -> 按具体场景
- 机制化 -> 形成固定做法
- 体系化 -> 形成系统方法

## Compression Moves

- Merge repeated clauses that state the same point.
- Replace "为了...我们将..." with direct action statements.
- Prefer active voice over passive or nominalized structures.
- Delete empty transitions like "基于上述背景" when no logic is lost.
- Convert long parallel slogans into one concrete sentence with subject + action + outcome.

## Keep-or-Replace Rule

- Keep domain terms if precision would be lost after replacement.
- Keep regulated vocabulary (legal, medical, compliance) and explain once in plain language.
- Keep user-defined names (product names, project names, team labels) unchanged.
